# 🛠️ CodeConnect Backend – Express.js + MongoDB + JWT

This is the **backend** for CodeConnect, a platform that helps developers find each other and build awesome projects together.

It handles authentication, profile management, filtering, and secure sessions using **JWT tokens** and **HTTP-only cookies**.

---



## ✨ Backend Features

- 🔐 JWT Authentication using **httpOnly cookies**
- 🧾 RESTful API endpoints for profiles and auth
- 📂 MongoDB schema for storing developer data
- 🧠 Mongoose + Express for structured logic
- 🧪 Simulated users for load testing and validation

---

## ⚙️ Local Setup Instructions

### 🧾 Prerequisites

- Node.js ≥ 14.x
- MongoDB Atlas or local MongoDB running
